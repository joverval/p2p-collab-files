📖 How it works

🖥️ Host

1. Enter your email and click Create Room

2. Click 📋 Copy invite link and share it with peers

3. When a peer requests to join, Approve or Reject

4. If manual mode: paste the peer's answer URL when prompted

👤 Peer

1. Open the invite link from the host

2. Enter your email and click Join Room

3. Wait for host approval

4. If manual mode: click 📋 Copy answer link and send it to the host
